CSS

.menu .title { background: black; color: white; font-weight: bold; border: none; padding: 5px; }   Formattazione per l'elemento di classe title nel menu

.menu ul{ margin: 0; padding: 0; list-style: none; border: none; } formattazione per le liste del menu: si resettano margini, padding, bordi e punti elenco

.menu li { margin: 0; } tutti i li nel menu hanno margine sinistro nullo

.menu li li { border-left: 5px solid blue; } i li nidificati in un altro li contenuti nel menu (cioè i li di livello maggiore di uno) hanno un bordo sinistro blu. Notate che, poichè ogni lista nidificata è posta all'interno di un elemento li, un elemento di terzo livello avrà, ad esempio, due bordi blu alla sia sinistra

.menu li a{ display: block; padding: 5px; border-left: 5px solid blue; border-bottom: 1px solid blue; text-decoration: none; color: white; width: 100%; } i link posti all'interno dei li del menu hanno, tra l'altro, un piccolo bordo bli a sinistra

.menu li li a{ font-size: 80%; } i link contenuti in li di levello maggiore di uno hanno una dimensione carattere dell'80% rispetto al contenitore

html>body .menu li a{ Non IE6 width: auto; } Trucco necessario per alcuni browser, che imposta a auto la larghezza dei link nel menu

.menu li a:hover{ border-left: 5px solid cyan; background-color: blue; color: white; } i link del menu, se sovrastati dal mouse, cambiano il colore del loro bordo sinistro
