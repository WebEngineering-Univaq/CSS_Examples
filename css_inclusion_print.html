Inclusione di CSS e media types

In questo esempio, la pagina HTML carica tre fogli di stile diversi.

  • Il primo (main.css) definisce delle proprietà comuni (in questo caso la dimensione del carattere), ed è sempre attivo.

  • Il secondo (screen.css) è un foglio di stile usato solo quando il media di visualizzazione è uno schermo, e determina il colore di fondo rosso per il testo nei paragrafi di classe1.

  • Il terzo foglio di stile (print.css) è un foglio di stile usato solo quando il media di visualizzazione è una stampante. In altre parole, print.css sostituisce screen.css quando la pagina viene stampata. In questo caso, il foglio di stile per la stampa si limita a colorare di blu lo sfondo dei paragrafi di classe1. Potete provare l'effetto semplicemente aprendo una anteprima di stampa di questa pagina.

L'uso di un foglio di stile alternativo per la stampa è una tecnica molto comune e di grande utilità. In un foglio di stile per la stampa è possibile, ad esempio, ridefinire le proprietà di stile per eliminare colori di sfondo e immagini che appesantirebbero inutilmente la stampa, o eliminare intere parti delle pagine (ad esempio i menu), per fare spazio ai contenuti, che sono l'unico oggetto utile in una stampa. Le proprietà utili ad ottenere questi effetti saranno illustrare più avanti.

Testo in paragrafo nidificato in un div di classe1

Head HTML

<link rel="stylesheet" media="all" type="text/css" href="main.css"/>

<link rel="stylesheet" media="screen" type="text/css" href="screen.css"/>

<link rel="stylesheet" media="print" type="text/css" href="print.css"/>

main.css

div.classe1 {font-size: 50px;}

screen.css

.classe1 {background-color: red}

print.css

.classe1 {color: blue}
