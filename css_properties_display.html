Uso della proprietà display

Mostrare dinamicamente elementi del documento

VISITALO! >>questo link visualizza del testo quando è attraversato da mouse<<

HTML

<a class="visitme" href="pippo.html"><span>VISITALO! &gt;&gt;</span>questo link visualizza del testo quando è attraversato da mouse<span>&lt;&lt;</span></a>

CSS

a.visitme span { display: none } gli span contenuti nei link di classe visitme non sono inizialmente visibili

a.visitme:hover span { display: inline } gli span contenuti nei link di classe visitme divengono visibili quando il mouse passa sopra al link

.menu ul li ul { display: none; border: 1px solid black; } le liste nidificate in altre liste all'interno dell'area del menu sono inizialmente invisibili

.menu ul li:hover ul { display: block; } le liste nidificate in altre liste all'interno dell'area del menu divengono visibili quando si passa il mouse sull'elemento (li) che le contiene
