CSS3: transizioni e opacità

Transizioni

La proprietà transition non è ancora supportata da tutti i browser allo stesso modo. Per Opera, Safari, Firefox e Chrome sono disponibili estenzioni specifiche con la stessa sintassi, mentre internet explorer non ha alcun supporto.

Opacità

La proprietà opacity non è supportata da internet explorer fino alle versione 8. Per ottenere un effetto simile, è necessario utilizzare la proprietà browser-specifica filter, con una diversa sintassi.

CSS

.opacity1, .opacity2 { background: blue; opacity:0.4; filter:alpha(opacity=40); }

.opacity1:hover, .opacity2:hover { opacity:1; filter:alpha(opacity=100); }

.opacity2 { transition: opacity 2s; -moz-transition: opacity 2s; -webkit-transition: opacity 2s; -o-transition: opacity 2s; }

HTML

<div class="opacity1">

<p>bla, bla, bla, bla, bla, bla, bla</p> <p>bla, bla, bla, bla, bla, bla, bla</p>

</div>
