CSS3: Bordi con immagini

Grazie alla proprieta' border-image, è possibile disegnare bordi molto complessi usando le immagini, ma senza ricadere nelle tecniche poco ortodosse utilizzate prima del supporto CSS.

E' necessario predisporre un'immagine che contenga gli otto elementi fondamentali di un bordo: quattro angoli e quattro lati.

L'immagine verrà poi "ritagliata" dai CSS ed utilizzata per comporre il bordo. Le parti usate per i lati potranno essere ripetute oppure deformate per coprire interamente il bordo.
