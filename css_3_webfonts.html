CSS3: Web fonts

Grazie alle web fonts, è possibile disporre di una vasta libreria di font, che estende notevolmente quella stabdard dei browser ed è soprattutto disponibile su ogni piattaforma.

Le web fonts vengono scaricati dinamicamente dai browser e messi nella cache, non installati nel sistema. Esistono vari siti che esportano web font gratuite, ma anche altri che mettono a disposizione migliaia di font a pagamento.

In questo esempio mostreremo come usare le Google Web Fonts. Il relativo codice CSS per la loro incorporazione si può trovare visitando la fond directory all'indirizzo https://www.google.com/webfonts

CSS

@font-face { font-family: 'Tangerine'; font-style: normal; font-weight: normal; src: local('Tangerine'), url('https://themes.googleusercontent.com/static/fonts/tangerine/v3/HGfsyCL5WASpHOFnouG-RD8E0i7KZn-EPnyo3HZu7kw.woff') format('woff'); } Carichiamo una web font da google associandola al nome 'Tangerine' nella sua versione regolare (non grassetto, non corsivo, ecc.). Notare l'uso di più versioni dell'attributo src per problemi di compatibilità crossbrowser.

@font-face { font-family: 'Tangerine2'; font-style: normal; font-weight: normal; src: local('Tangerine'), url('https://themes.googleusercontent.com/static/fonts/tangerine/v3/HGfsyCL5WASpHOFnouG-RD8E0i7KZn-EPnyo3HZu7kw.woff') format('woff'); } @font-face { font-family: 'Tangerine2'; font-style: normal; font-weight: bold; src: local('Tangerine Bold'), local('Tangerine-Bold'), url('https://themes.googleusercontent.com/static/fonts/tangerine/v3/UkFsr-RwJB_d2l9fIWsx3nhCUOGz7vYGh680lGh-uXM.woff') format('woff'); } Possiamo anche caricare lo stesso font con un nome diverso, e/o associare font diversi a stili diversi della stessa famiglia

@font-face { font-family: 'Aclonica'; font-style: normal; font-weight: normal; src: local('Aclonica'), local('Aclonica-Regular'), url('https://themes.googleusercontent.com/static/fonts/aclonica/v3/VVH16EjBQLZ2NYB8eVnrO_esZW2xOQ-xsNqO47m55DA.woff') format('woff'); }

.google1 { font-family: 'Tangerine',Times,serif; font-size: 20pt; } .google2 { font-family: 'Tangerine2',Times,serif; font-size: 20pt; } .google3 { font-family: 'Aclonica',Helvetica,sans-serif; font-size: 20pt; } .google4 { font-family: 'Ribeye Marrow', cursive; font-size: 20pt; } Associamo quindi le nuove famiglie di font a qualsiasi elemento tramite le normali proprietà CSS. Non dimentichiamo di offrire delle alternative locali.
