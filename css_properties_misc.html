Proprietà content e counter

ciao
riciao

HTML

<div class="contato">ciao</div>

<div class="contato">riciao</div>

CSS

.contato:before {content: "Contatore " counter(contatore1) ". ";} prima di ogni blocco di classe contato viene posta la scritta "Contatore" seguita dal volare del contatore1 e da un punto

.contato {counter-increment: contatore1;} ogni blocco di classe contato incrementa di uno il contatore1

<h1>Titolo1</h1>

CSS

h1:before {content: counter(hl1) ". ";}

h1 {counter-increment: hl1; counter-reset: hl2 0; }

h2:before {content: counter(hl1) "." counter(hl2) ". ";}

h2 {counter-increment: hl2;}

le regole relative agli elementi H1 e H2 gestiscono due contatori. In particolare, la regola H1 resetta a zero il contatore usato per H2
