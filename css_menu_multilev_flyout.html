CSS

.menu .title { background: black; color: white; font-weight: bold; border: none; padding: 5px; }

.menu ul{ margin: 0; padding: 0; list-style: none; border: none; background: navy; width: 200px; }

.menu li { margin: 0; position: relative; } Gli item del menu vengono impostati con un posizionamento relativo "fittizio" per renderli origine degli elementi posizionati al loro interno

.menu li a{ display: block; padding: 5px; text-decoration: none; color: white; position: relative; border-left: 5px solid blue; border-bottom: 1px solid blue; }

.menu li a:hover{ background-color: blue; border-left: 5px solid cyan; color: white; }

/*setting base dei sottomenu*/ .menu ul ul { visibility:hidden; position:absolute; top: 0; left:200px; } le liste nidificate (ul dentro ul) sono rese invisibili e posizionate in maniera assoluta in alto e 200 pixel a sinistra del loro contenitore (che è la voce di menu che le contiene)

hover sul primo livello: mostrare secondo livello .menu ul :hover ul { visibility:visible; } quando il mouse è su un elemento di una lista, rendiamo visibili tutte le liste in essa nidificate

/*hover sul secondo livello: mostrare terzo livello*/ .menu ul :hover ul :hover ul{ visibility:visible; }

/*hover sul terzo livello: mostrare quarto livello*/ .menu ul :hover ul :hover ul :hover ul { visibility:visible; }

/*nasconde i (due) livelli di menu successivi a quello visualizzato*/ .menu ul :hover ul ul{ visibility:hidden; } quando mostriamo il secondo livello, nascondiamo tutti i successivi

.menu ul :hover ul :hover ul ul{ visibility:hidden; }

.menu table {position:absolute; border-collapse:collapse; top:0; left:0; z-index:100; font-size:1em;} Completiamo il "trucco" per Internet Explorer
