Nuove proprietà dei CSS versione 3

Bordi arrotondati con i CSS versione 3

In questo esempio gli angoli smussati sono ottenuti usando le proprietà di sile apposite dei CSS3.

Purtroppo non tutti i browser supportano ancora queste proprietà con il loro nome "ufficiale", quindi per compatibilità è utile impostare sempre anche alcune proprietà "proprietarie" apprtenenti alle estensioni dei singoli browser.

Se poi, come accade in molte versioni recenti di IE, i CSS3 non fossero ancora supportati, in ogni caso il bordo degraderebbe bene perdendo solo la ritondità degli angoli.

CSS

.box { border:1px solid #8AAFE1; } Impostiamo prima di tutto il bordo con le proprietà standard

.round { -moz-border-radius:5px; -webkit-border-radius:5px; border-radius:5px; } Quindi procediamo con la curvartura. Nell'ordine troviamo la proprietà interna di mozilla, quella di Opera (e Chrome), e quella ufficiale CSS3

<div class="box round">
<p>bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla</p>
