CSS

a[accesskey] {border-left: 2px solid #c0c0c0; border-top: 2px solid #c0c0c0; border-right: 2px solid #909090; border-bottom: 2px solid #909090; background: #f0f0f0; padding: 2px; } tutti i tag a con attributo accesskey sono formattati in maniera speciale

a[rel~=index] {font-style: italic;} tutti i tag a il cui attributo rel contiene index sono formattati in corsivo

a[rel=index] {font-weight: bold;} tutti i tag a con attributo rel uguale a index sono formattati in grassetto

input[type=password] {border-style: dotted;} tutti gli input password hanno un bordo puntinato (oltre che blu!)

Selettori di classe e pseudo classe

link con hover effect

HTML

<p><a class="hovering" href="pippo.html">link con hover effect</a></p>

<p><input type="text" class="activating" name="pippo" value="input con hover e focus effect"/>

CSS

a.hovering {color:black} gli elementi a di classe hovering hanno testo nero

a.hovering:hover {color:blue} gli elementi a di classe hovering, quando il mouse passa su di essi, hanno testo di colore blu

input.activating {border:2px solid black} gli input di classe activating hanno un bordo nero spesso

input.activating:hover {border-color:blue} ...che diventa blu quando il mouse è sopra al controllo

input.activating:focus {border-color:red} ...e rosso se il cursore è nel controllo (focus)

HTML

<div class="diffbold">testo <b>in grassetto</b> in una div di classe diffbold</div>

<div>testo <b>in grassetto</b> in una div senza classe</div>

<div class="diffitalic">testo <i>in corsivo</i> e altro testo <span>in uno span <i>sempre in corsivo</i></span></div>

<h1>titolo 1</h1>

<h3>titolo 3</h3>

<h3>titolo 3</h3>

CSS

.diffbold b {letter-spacing: 2px;} gli elementi b all'interno di elementi di classe diffbold hanno testo con lettere spaziate di 2 pixel

.diffitalic > i {letter-spacing: -2px;} gli elementi i che sono figli diretti di elementi di classe diffitalic hanno testo con lettere ravvicinate di 2 pixel

h1 + h3 {margin-top:-10px; color: blue;} gli elementi h3 che seguono immediatamente un elemento h1 sono di colore blu e hanno un margine superiore negativo di 10 pixel (sono cioè avvicinate all'elementi h1 che li precede)
