Nuove proprietà dei CSS versione 3

Ombre con i CSS versione 3

Un'altra proprietà introdotta dai CSS3 e utile per "abbellire" i propri riquadri è quella che permette di generare ombre.

Anche in questo caso, per ottenere lo stesso risultato prima ci si avvaleva di tecniche alternative basate su immagini.

bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla

bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla

CSS

.box { border:1px solid #8AAFE1; } Impostiamo prima di tutto il bordo con le proprietà standard.

.shadow { -moz-box-shadow:5px 5px 2px #DDD; -webkit-box-shadow:5px 5px 2px #DDD; box-shadow:5px 5px 2px #DDD; } Quindi procediamo con l'ombra. Nell'ordine troviamo la proprietà interna di mozilla, quella di Opera (e Chrome), e quella ufficiale CSS3 I valori della proprietà sono: lo scostamento orizzontale e verticale dell'ombra (anche negativi), il raggio di sfumatura e il colore.

HTML

<div class="box shadow">
<p>bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla</p>
<p>bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla</p>
</div>

Nell'ordine troviamo la proprietà interna di mozilla, quella di Opera (e Chrome), e quella ufficiale CSS3 I valori della proprietà sono: lo scostamento orizzontale e verticale dell'ombra (anche negativi), il raggio di sfumatura e il colore.

Ombre per il testo con i CSS versione 3

Le ombre si possono applicare anche al testo, ma con una proprietà differente.

bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla

bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla

CSS

.textshadow { text-shadow:2px 2px 0 #AAA; } I valori della proprietà sono: lo scostamento orizzontale e verticale dell'ombra (anche negativi), il raggio di sfumatura e il colore.

HTML

<div class="textshadow">
<p>bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla</p>
<p>bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla, bla</p>
</div>
